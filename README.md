# E学圈 - 视频剪辑学习平台

E学圈官网是一个专注于视频剪辑教学的产品展示页面，采用纯HTML/CSS/JavaScript开发。

## 项目结构

```
official/
├── index.html          # 首页（E学圈产品展示）
├── about.html          # 关于我们（已注释）
├── services.html       # 产品服务（已注释）
├── news.html          # 新闻动态（已注释）
├── contact.html       # 联系我们（已注释）
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # 交互脚本
└── README.md          # 说明文档
```

## 页面内容

### 首页结构
1. **金刚区**：公司简介，突出教育科技定位
2. **产品展示区**：
   - **左侧**：E学圈应用介绍，包含核心功能说明和下载按钮
   - **右侧**：手机模型展示应用界面效果

### E学圈应用特色
- 🎬 **系统教学**：从基础到进阶的结构化视频剪辑教学
- 🔄 **互动分享**：学员作品分享交流平台
- 🎯 **新手友好**：零基础也能快速上手

## 主要特性

- 📱 响应式设计，完美适配桌面和移动端
- 🎨 现代化UI设计，突出产品特色
- 📱 精美的手机模型展示界面
- ⚡ 纯原生代码，无外部依赖

## 快速开始

### 方法一：直接打开
直接在浏览器中打开 `index.html` 文件即可预览。

### 方法二：使用本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 然后访问 http://localhost:8000
```

## 自定义修改

### 修改应用信息
在 `index.html` 中搜索并替换：
- 应用名称：E学圈
- 应用介绍和功能描述
- 联系方式和公司信息

### 修改界面展示
在手机模型 `.app-interface` 区域可以修改：
- 应用界面内容
- 课程列表
- 播放器样式

### 修改颜色主题
在 `css/style.css` 中修改CSS变量：
```css
:root {
    --primary-color: #2563eb;      /* 主色调 */
    --secondary-color: #1e40af;    /* 次要色 */
    --text-color: #333;            /* 文本颜色 */
}
```
